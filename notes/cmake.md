# cmake

> generator of buildsystems - can produce `Makefile`, `ninja` build files, etc. from same `CMakeLists.txt`

## usage
```sh
cmake -version

cmake .
```

## CMakeLists.txt
```txt
cmake_minimum_required(VERSION 2.8.9)
project (hello)
add_executable(hello helloworld.cpp)
```
