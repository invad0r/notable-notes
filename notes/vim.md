# vim

## usage
```sh
vim -p file1 file2 file3	  # open files in tabs

vim $(!!)		                # open file from last output
```

## vim file
```sh
# -*- mode: ruby -*-
# vi: set ft=ruby : 

# vim: ft=json
```
