# scutil

> Manage system configuration parameters on macos

## usage

```sh
scutil --get HostName
scutil --get LocalHostName
scutil --get ComputerName


scutil --dns
# or
cat /etc/resolv.conf


grep nameserver <(scutil --dns)     # list dns cache

tail -f /private/var/log/system.log

sudo killall -INFO mDNSResponder
```
