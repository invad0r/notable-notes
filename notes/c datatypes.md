# c datatypes


## usage
```c
int /* 2/4%d*/

char/* 1%c */

float/* 4%f */

double/* 8%lf */

short int /* 2 %hd */

unsigned int /* 2/4%u*/

long int /* 4/8%li*/

long long int /* 8 %lli */

unsigned long int /* 4%lu */

unsigned long long int /* 8%llu */

signed char/* 1%c */

unsigned char/* 1%c */

long double /* 0/12 or 16%Lf*/
```
