# more

> file perusal filter for crt viewing 

## usage
```sh
more -V             # get version

more --help        # show options

more -d file      # prompts `"[Press space to continue, 'q' to quit.]"`

more -s file      # squeeze multiple blank lines into one. 

# space     scroll down
# b         scroll back/up
# [n]k      scoll back n lines
# v         start vi

```
